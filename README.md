# 🔐 GitHub Authentication Setup

This repository provides a step-by-step guide for configuring secure GitHub authentication using **SSH keys** and **Personal Access Tokens (PAT)**.

---

## 📌 Why This Matters

Using secure authentication methods avoids repeated login prompts, protects your credentials, and enables smooth Git operations like cloning, pushing, and pulling.

GitHub recommends:
- **SSH authentication** for daily development use
- **Personal Access Tokens (PAT)** for HTTPS and CI/CD pipelines
